# Мобы

### Описание <a href="#opisanie" id="opisanie"></a>

Помимо новых предметов на сервере есть мобы с уникальным для них дропом. Существует 4 уровня мобов: Т1, Т2, Т3, Т4. Каждому уровню мобов соответствует свой уровень лута. С мобов более высоких уровней дроп более высокого уровня, но этих мобов можно убить только оружием на 1 тир ниже или своего тира

Например, Т3 Зомби можно убить Т2 мечом, но убивать его придется долго ибо меч очень плохо пробивает его. Т1 меч же вообще не пробивает броню Т3 Зомби ибо очки его брони выше чем урон меча.

У всех мобов есть шанс дропа ванильных предметов

| Ванильный предмет | Шанс выпадения (%) |
| ----------------- | ------------------ |
| Кости             | 10%                |
| Сахар             | 10%                |
| Порох             | 25%                |
| Картофель         | 10%                |
| Морковь           | 10%                |
| Уголь             | 10%                |
| Нитки             | 10%                |
| Слизь             | 2%                 |
