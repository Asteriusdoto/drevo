# Совет Восьми

| Требуемая технология | **Совет Восьми (Инновации)**                  |
| -------------------- | --------------------------------------------- |
| Стоимость            | **10.000.000 монет**                          |
| Размер               | **4х4**                                       |
| Налог                | **-/сутки**                                   |
| Молоточки            | &#x32;**.000.000**                            |
| Прочность            | **5.000 (Регенерация 6 прочности в секунду)** |
| Очки                 | **200.000**                                   |
| Ограничение (кол-во) | **1**                                         |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### Модификации и эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Пушки наносят в 2 раза меньше урона по чудесам.
* Совет Восьми невозможно сломать пушками.
* На чудеса света цивилизации в которой построен совет восьми не работает зачарование панчаут и любые другие баффы на доп урон по постройкам
* Раз в день (во время сбора налогов 20:00 мск) дает очки в размере 20.000\* (эру цивилизации + 1). Данные очки не снимаются со счета цивилизации даже если чудо сломали
