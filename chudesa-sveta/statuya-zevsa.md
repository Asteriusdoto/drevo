# Статуя Зевса

| **Требуемая технология** | **Статуя Зевса (Механизмы + Металлургия)**   |
| ------------------------ | -------------------------------------------- |
| Стоимость                | **800,000 монет**                            |
| Размер                   | **4х4**                                      |
| Налог                    | **-/сутки**                                  |
| Молоточки                | **350,000**                                  |
| Прочность                | **5000 (Регенерация 6 прочности в секунду)** |
| Очки                     | **70000**                                    |
| Ограничение (кол-во)     | **1 (на Мир)**                               |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### Модификации и эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

Башня тесла: <mark style="color:$warning;">**+2 к урону**</mark>

<mark style="color:purple;">**+20 культуры**</mark> в час за каждую военную башню, военный корабль, капитолий и ратушу в час в каждый город цивилизации

В бою есть <mark style="color:$warning;">**5% шанс**</mark> что противник получит молнию на 3 урона и поджог

Работает как башня тесла радиусом 350 блоков и имеет 8 урона.
