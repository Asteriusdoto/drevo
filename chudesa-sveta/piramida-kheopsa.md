# Пирамида Хеопса

| Требуемая технология | **Пирамида Хеопса (Мореплавание + Архитектура + Механизмы)** |
| -------------------- | ------------------------------------------------------------ |
| Стоимость            | **750,000 монет**                                            |
| Размер               | **4х4**                                                      |
| Налог                | -/сутки                                                      |
| Молоточки            | **550,000**                                                  |
| Прочность            | **5000 (Регенерация 6 прочности в секунду)**                 |
| Очки                 | **85,000**                                                   |
| Ограничение (кол-во) | **1 (на мир)**                                               |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### Модификации и эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Культура (на город): <mark style="color:purple;">**+25%**</mark>
* <mark style="color:$success;">**10%**</mark> дохода с коттеджа идёт в качестве <mark style="color:purple;">**культуры**</mark> в город, в котором построен коттедж. Распространяется на все коттеджи цивилизации
* Кузница дает 1.75 слитка вместо 1.25 при переплавке
* Инструменты (Кирка, лопата, топор) не теряют прочность
