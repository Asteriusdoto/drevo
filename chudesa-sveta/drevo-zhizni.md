# Древо жизни

| Размер               | **4х4**                                                     |
| -------------------- | ----------------------------------------------------------- |
| Требуемая технология | **Древо Жизни (Продуктивность + Загонная охота+Коммерция)** |
| Стоимость            | **800,000 монет**                                           |
| Налог                | **-/сутки**                                                 |
| Молоточки            | **575,000**                                                 |
| Прочность            | **5000 (Регенерация 6 прочности в секунду)**                |
| Очки                 | **70000**                                                   |
| Ограничение (кол-во) | **1 (на Мир)**                                              |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### Модификации и эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Торговые посты/лодки не занимают слоты города (Работает на все города Цивилизации)
* Торговые ресурсы дают <mark style="color:blue;">+60 науки</mark> (Работает на все города Цивилизации)
* Стоимость зданий занимающих слоты (на Цивилизацию): <mark style="color:$success;">**-20%**</mark>
* Количество слотов улучшений (на Город): <mark style="color:$success;">**+100%**</mark>
