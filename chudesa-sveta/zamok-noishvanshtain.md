# Замок Нойшванштайн

| **Требуемая технология** | **Замок Нойшванштайн (Дубление + Химия + Булатная сталь + Археология)** |
| ------------------------ | ----------------------------------------------------------------------- |
| Стоимость                | **4,500,000 монет**                                                     |
| Размер                   | 3х9                                                                     |
| Налог                    | **-/сутки**                                                             |
| Молоточки                | **2,000,000**                                                           |
| Прочность                | **3,500 (Регенерация 6 прочности в секунду)**                           |
| Очки                     | **130,000**                                                             |
| Ограничение (кол-во)     | **1 (на Мир)**                                                          |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### Модификации и эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Защита всех жителей (на Цивилизацию): <mark style="color:$success;">**+0,5**</mark>
* Стоимость починки в бараках (на Цивилизацию): <mark style="color:$success;">**-25%**</mark>
* Радиус стрельбы башен пушкарей (на Цивилизацию): <mark style="color:$success;">**+50%**</mark>
* Урон башен лучников (на Цивилизацию): <mark style="color:$warning;">**+2**</mark> (Действует пока не уничтожен)
* Увеличивает длительность подсветки всех кто был <mark style="color:$warning;">**обнаружен разведкой**</mark> цивилизации до 10 секунд. (Действует пока не уничтожен)
