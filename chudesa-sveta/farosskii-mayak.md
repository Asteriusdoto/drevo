# Александрийский маяк

| Требуемая технология | **Александрийский маяк (Экономика + Инженерия)** |
| -------------------- | ------------------------------------------------ |
| Стоимость            | **800,000 монет**                                |
| Размер               | **4х4**                                          |
| Налог                | **- /сутки**                                     |
| Молоточки            | **600,000**                                      |
| Прочность            | **5000 (Регенерация 6 прочности в секунду)**     |
| Очки                 | **70.000**                                       |
| Ограничение (кол-во) | **1 (на Мир)**                                   |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### **Модификациии и эффекты** <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Радиус стрельбы всех башен (на Цивилизацию): <mark style="color:red;">**+50%**</mark>
* Прибыль торговых кораблей (на Цивилизацию): <mark style="color:green;">**+20%**</mark>
* Каждый чанк океана (любого) дает <mark style="color:blue;">+0,4 науки</mark> и +<mark style="color:$warning;">0,4 молотка</mark> (Работает на все города Цивилизации)
