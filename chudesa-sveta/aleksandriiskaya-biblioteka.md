# Александрийская библиотека

| Размер               | 4х4                                                  |
| -------------------- | ---------------------------------------------------- |
| Требуемая технология | **Александрийская библиотека (Инженерия + Алхимия)** |
| Стоимость            | **500,000 монет**                                    |
| Налог                | **- /сутки**                                         |
| Молоточки            | **500,000**                                          |
| Прочность            | **5,000 (Регенерация 6 прочности в секунду)**        |
| Очки                 | 70000                                                |
| Ограничение (кол-во) | **1 (на мир)**                                       |

***

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

Позволяет членам цивилизации, которая владеет Александрийской библиотекой, приобрести 4 эксклюзивных зачарования стоимость 25.000 монет каждое:

* <mark style="color:red;">**Заговор огня II**</mark>
* <mark style="color:$warning;">**Огнеупорность III**</mark>
* <mark style="color:red;">**Горящая стрела I**</mark>
* <mark style="color:blue;">**Панчаут**</mark> (дополнительный случайный урон по зданиям от 1 до 5)

***

### **Модификации и эффекты** <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Наука <mark style="color:blue;">+15%</mark> (на Цивилизацию)
* Каждая построенная [библиотека](../osnovnye-sooruzheniya/antichnost/biblioteka.md) дает <mark style="color:blue;">+100 науки</mark> (на Цивилизацию)
