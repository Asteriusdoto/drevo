# Нотр-Дам

| Размер               | 3**х8**                               |
| -------------------- | ------------------------------------- |
| Требуемая технология | **Нотр-Дам-де-Пари (Машиностроение)** |
| Стоимость            | **1,750,000 монет**                   |
| Налог                | **-/сутки**                           |
| Молоточки            | **850,000**                           |
| Прочность            | **12000**                             |
| Очки                 | **120000**                            |
| Ограничение (кол-во) | **1 (на мир)**                        |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### Модификации и эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Переход в новый строй длится в 2 раза меньше по времени
* Цивилизация не падает в анархию при переходе в новый строй и максимальное время перехода 2 часа.
* Приносит <mark style="color:$warning;">**15,000 монет**</mark> в сутки за каждый мирный город на сервере
* В случае объявления войны цивилизации в которой построен Нотр-Дам, цивилизация агрессор получает <mark style="color:orange;">**двойные штрафы по налогу от войны**</mark>
* В случае объявления войны цивилизации, у которой построен Нотр-Дам цивилизация агрессор получает <mark style="color:red;">+2.5 несчастья</mark> в каждый город.
