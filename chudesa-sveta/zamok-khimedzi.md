# Замок Химедзи

| Требуемая технология | **Замок Химедзи (Эффективность + Артиллерия + Ковка мечей)** |
| -------------------- | ------------------------------------------------------------ |
| Стоимость            | **1,450,000 монет**                                          |
| Размер               | **4х4**                                                      |
| Налог                | **-/сутки**                                                  |
| Молоточки            | **750,000**                                                  |
| Прочность            | **5000 (Регенерация 6 прочности в секунду)**                 |
| Очки                 | **115,000**                                                  |
| Ограничение (кол-во) | **1 (на Мир)**                                               |

{% hint style="danger" %}
**Внимание! Вы можете построить только 2 чуда света от одного города!**
{% endhint %}

***

### Модификации и эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

* Каждая построенная Башня и корабль лучников дают <mark style="color:$warning;">**+1 хп**</mark> контрольным блокам города, в котором построена башня или корабль (Бафф работает во всех городах Цивилизации)
* Позволяет зачаровать Невесомость 4 на ботинки
* Броня и оружие не тратят прочность при смерти (Работает на все города Цивилизации)
* +0,5 к урону с меча и лука (Работает на все города Цивилизации)
