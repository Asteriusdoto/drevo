# Стена

| Размер               | **-**                        |
| -------------------- | ---------------------------- |
| Необходимое здание   | **-**                        |
| Требуемая технология | **Каменная кладка**          |
| Стоимость            | **1,000 (за сегмент) монет** |
| Налог                | **500 (за сегмент) /сутки**  |
| Молоточки            | **-**                        |
| Прочность            | **1,000**                    |
| Очки                 | **0**                        |
| Ограничение (кол-во) | **Неограниченно**            |

### Описание <a href="#opisanie" id="opisanie"></a>

Стена - уникальный тип зданий, она не требует молотков и времени. Для постройки необходимо установить <mark style="color:$warning;">2 точки</mark>. Через эти 2 точки будет проведена стена. Стена защищает блоки под собой, ломать могут только лидеры.

{% hint style="info" %}
Все блоки под стеной будут структурно защищены ее прочностью. Учитывайте это при постройке дефа.
{% endhint %}
