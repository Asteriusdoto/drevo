# Капитолий

| Размер               | **4х4**                                  |
| -------------------- | ---------------------------------------- |
| Необходимое здание   | -                                        |
| Требуемая технология | -                                        |
| Стоимость            | **Для постройки требуется 125000 монет** |
| Налог                | **0 /сутки**                             |
| Молоточки            | 200                                      |
| Прочность            | **8 контрольных блоков(8x100 HP)**       |
| Очки                 | 1**5000**                                |
| Ограничение (кол-во) | **1**                                    |

### Обзор <a href="#obzor" id="obzor"></a>

Капитолий является <mark style="color:$success;">столицей</mark> вашей Цивилизации. Капитолий - это основа для дальнейшего развития всей Цивилизации. Без него вы не сможете ничего изучать и строить.

***

### Прочность контрольных блоков <a href="#prochnost_kontrolnykh_blokov" id="prochnost_kontrolnykh_blokov"></a>

Контрольных блоков в Капитолии <mark style="color:yellow;">8</mark>, их прочность равна <mark style="color:yellow;">100</mark> и может быть увеличена Чудесами или другими баффами

***

### Специальные функции <a href="#specialnye_funkcii" id="specialnye_funkcii"></a>

Капитолий - это первое здание, которое вы строите при основании Цивилизации. В Капитолии имеются несколько кроватей, печек, сундуков, которые нельзя сломать в мирное время. Также Капитолий имеет и оборонительные сооружения: <mark style="color:yellow;">4 структурных</mark> [<mark style="color:yellow;">Башни Лучников</mark>](../../voennye-bashni-i-korabli/bashni/bashnya-luchnikov.md)<mark style="color:yellow;">.</mark> Также в Капитолии располагается <mark style="color:$danger;">Военная комната</mark>, куда во время войны попадают все игроки после смерти.

***

### Капитолий во время войны <a href="#kapitolii_vo_vremya_voiny" id="kapitolii_vo_vremya_voiny"></a>

В Цивилизации можно построить только 1 капитолий, не включая тот фактор, что во время войны Вы можете захватить другую Цивилизацию и у Вас станет 2 Капитолия.

Капитолий играет <mark style="color:$danger;">ключевую роль</mark> во время войны. Чтобы захватить Цивилизацию врага, необходимо <mark style="color:$danger;">захватить Капитолий</mark>, что будет очень непросто!

***

### Захват Капитолия <a href="#zakhvat_kapitoliya" id="zakhvat_kapitoliya"></a>

Когда враг захватил Ваш Капитолий, Вы <mark style="color:$warning;">становитесь частью его Цивилизации.</mark> Подробнее об этом смотрите в разделах [Война](../../upravlenie-gorodom-i-civilizaciei/diplomatiya/voina.md)
