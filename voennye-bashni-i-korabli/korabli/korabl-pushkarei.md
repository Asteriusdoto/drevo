# Корабль пушкарей

| Размер               | **2х1**                |
| -------------------- | ---------------------- |
| Необходимое здание   | **Верфь**              |
| Требуемая технология | **Баллистика**         |
| Стоимость            | **175,000 монет**      |
| Налог                | **3,500 монет /сутки** |
| Молоточки            | **7,500**              |
| Прочность            | **1,250**              |
| Очки                 | **2000**               |
| Ограничение (кол-во) | **1**                  |

### Обзор <a href="#obzor" id="obzor"></a>

Корабль Пушкарей является аналогом [Башни Пушкарей ](../bashni/bashnya-pushkarei.md)на суше.

* Стреляет ядром, которое поражает цель, поджигая ее!
* Дальность: 125 блоков
* Минимальное расстояние между башней пушкарей и кораблем пушкарей 64 блока, между кораблем пушкарей и кораблем пушкарей 96 блоков (остальные типы башен и кораблей при расчёте расстояния не учитываются)
* Расстояние теперь учитывается только по X и Y
* Урон от выстрела: 7 урона
* Скорость атаки: 1 залп каждые 5 секунды
