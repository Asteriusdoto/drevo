# Корабль разведчиков

| Размер               | **2х1**              |
| -------------------- | -------------------- |
| Необходимое здание   | **Верфь**            |
| Требуемая технология | **Мореплавание**     |
| Стоимость            | **22,500 монет**     |
| Налог                | **800 монет /сутки** |
| Молоточки            | **1,750**            |
| Прочность            | **500**              |
| Очки                 | **1000**             |
| Ограничение (кол-во) | **4**                |

Корабль Разведчиков - структура, которая показывает координаты врагов в пределах культуры Вашей Цивилизации. Является аналогом [Башни разведчиков ](../bashni/bashnya-razvedchikov.md)на суше.

Дальность: 400 Блоков (Квадрат со стороной 400, Корабль Разведчиков в центре квадрата). Скорость Разведки: Нужно использовать команду <mark style="color:$warning;">/town set scoutrate</mark> 10-60 секунд. Изначально: 30 секунд

***

### Отображение местоположения врагов <a href="#otobrazhenie_mestopolozheniya_vragov" id="otobrazhenie_mestopolozheniya_vragov"></a>

Отображение происходит с помощью вывода в чат сообщений с координатами нахождения Вашего врага или преступника. Пример:

{% hint style="warning" %}
Башня разведчиков засекла игрока Asterius\_doto <mark style="color:$danger;">(HOSTILE)</mark> на координатах (x, y, z) в городе Spawn.
{% endhint %}

{% hint style="warning" %}
Башня разведчиков засекла игрока who\_dunnit <mark style="color:$danger;">(OUTLAW)</mark> на координатах (x, y, z) в городе Spawn.
{% endhint %}
