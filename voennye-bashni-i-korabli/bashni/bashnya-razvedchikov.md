# Башня разведчиков

| Размер               | **1х1**              |
| -------------------- | -------------------- |
| Необходимое здание   | **-**                |
| Требуемая технология | **Кладка**           |
| Стоимость            | **22,500 монет**     |
| Налог                | **800 монет /сутки** |
| Молоточки            | **1,750**            |
| Прочность            | **500**              |
| Очки                 | **1000**             |
| Ограничение (кол-во) | **4**                |

### Обзор <a href="#obzor" id="obzor"></a>

Башня Разведчиков - военная структура, которая показывает координаты (x,y,z) врагов в пределах культуры Вашей Цивилизации. Тем самым Вы можете найти врага и убить его. Также можно будет понять, с какой стороны враги собираются нападать.

* Дальность: 400 Блоков (Квадрат со стороной 400, Башня Разведчиков в центре квадрата).
* Скорость Разведки: Нужно использовать команду <mark style="color:$warning;">/town set scoutrate</mark> 1-60 секунд. Изначально: 30 секунд
* Нельзя ставить ниже 63 высоты

***

### Отображение местоположения врагов <a href="#otobrazhenie_mestopolozheniya_vragov" id="otobrazhenie_mestopolozheniya_vragov"></a>

Отображение происходит с помощью вывода в чат сообщений с координатами нахождения Вашего врага или преступника. Пример:

{% hint style="warning" %}
Башня разведчиков засекла игрока Asterius\_doto <mark style="color:$danger;">(HOSTILE)</mark> на координатах (x, y, z) в городе Spawn.
{% endhint %}

{% hint style="warning" %}
Башня разведчиков засекла игрока who\_dunnit <mark style="color:$danger;">(OUTLAW)</mark> на координатах (x, y, z) в городе Spawn.
{% endhint %}
