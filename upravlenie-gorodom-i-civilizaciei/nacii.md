# Нации

Нация - уникальный выбор вашей цивилизации, который можно сделать в начале игры. Каждая нация обладает уникальными характеристиками. Нации доступны к выбору сразу после постройки капитолия.&#x20;

{% hint style="danger" %}
&#x20;**ВЫБОР НАЦИИ ДОСТУПЕН ОДИН РАЗ ЗА СЕЗОН** и её изменение по ходу игры невозможно.
{% endhint %}

{% hint style="info" %}
Выбрать нацию вы можете командой /civ nation change "Название нации". Выбрать нацию может ТОЛЬКО основатель цивилизации.
{% endhint %}
