### Команды управления группами цивилизации <a href="#komandy_upravleniya_gruppami_civilizacii" id="komandy_upravleniya_gruppami_civilizacii"></a>

| Команда           | Аргумент                  | Описание                                            |
| ----------------- | ------------------------- | --------------------------------------------------- |
| /civ group add    | <имя> \<leaders/advisers> | Назначает жителя <имя> в группу \<leaders/advisers> |
| /civ group remove | <имя> \<leaders/advisers> | Удаляет жителя <имя> из группы \<leaders/advisers>  |
| /civ group info   | \<leaders>                | Показывает списки лидеров и советников              |

Название стандартных групп цивилизаций:

* Овнер - владелец цивилизации, может быть только один в одно время и не может быть удален из групп или кикнуть из города.
* Лидеры - люди имеющие права внутри цивилизации
* Советники - люди которые заменяют лидеров, могут ставить технологии на изучения но не больше.
