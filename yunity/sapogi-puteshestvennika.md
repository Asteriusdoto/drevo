# Сапоги путешественника

Сапоги путешественника - это особый артефакт, который наделяет владельца эффектом <mark style="color:blue;">Скорость II</mark>. Его могут использовать только жители цивилизации, где был сделан юнит. Не выпадает при смерти. Тренируется в [Бараках](../osnovnye-sooruzheniya/drevnii-mir/baraki.md).

| Стоимость (Деньги)     | **125,000**        |
| ---------------------- | ------------------ |
| Стоимость (Молоточки)  | **4,500**          |
| Необходимая технология | **Пивоварение**    |
| Тип предмета           | **Козий рог**      |
| Лимит                  | **Не ограничено**  |
