# Тележка с едой

Тележка с едой - это юнит обеспечения, который избавляет игроков от необходимости постоянно следить за шкалой голода и тратить время на употребление пищи. Его могут использовать только жители цивилизации, где был сделан юнит. Не выпадает при смерти. Тренируется в [Бараках](../osnovnye-sooruzheniya/drevnii-mir/baraki.md).

| Стоимость (Деньги)    | **275,000**                    |
| --------------------- | ------------------------------ |
| Стоимость (Молоточки) | **9,000**                      |
| Тип предмета          | **Бамбуковый плот с сундуком** |
| Ограничение (лимит)   | **Не ограничено**              |
