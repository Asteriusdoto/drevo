# Лаборатория

| Размер               | **1х1**               |
| -------------------- | --------------------- |
| Необходимое здание   | **Школа**             |
| Требуемая технология | **Пластмассы**        |
| Стоимость            | **100,000 монет**     |
| Налог                | **2,000 монет/сутки** |
| Молоточки            | **10,000**            |
| Прочность            | **1,000**             |
| Очки                 | 8500                  |
| Ограничение (кол-во) | **1**                 |

### Описание <a href="#opisanie" id="opisanie"></a>

Лаборатория - это структура поздних этапов игры, которая генерирует [науку](../../osnovnye-kharakteristiki/nauka/) для орода, а также снижает стоимость изучений в монетах. Лабораторию можно построить, только если в городе уже построена [Школа](../era-renessansa/shkola.md).

***
