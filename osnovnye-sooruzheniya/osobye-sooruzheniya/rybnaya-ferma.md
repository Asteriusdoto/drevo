# Рыбная ферма

| Размер               | **1х1**               |
| -------------------- | --------------------- |
| Необходимое здание   | **-**                 |
| Требуемая технология | **Механизмы**         |
| Стоимость            | **25,000 монет**      |
| Налог                | **5000 монет /сутки** |
| Молоточки            | **1200**              |
| Прочность            | **10**                |
| Очки                 | **300**               |
| Ограничение (кол-во) | **Неограничено**      |

### Описание <a href="#opisanie" id="opisanie"></a>

Рыбная ферма - это постройка, которая будет вылавливать рыбу и меркурий, используя удочки. Тип рыбы зависит от уровня структуры. После поимки рыба складируется в двойной сундук. Если он будет полон, работа будет прекращена до освобождения места.

***

### Особенности <a href="#osobennosti" id="osobennosti"></a>

* Занимает слот города

***

### Как работает структура? <a href="#kak_rabotaet_struktura" id="kak_rabotaet_struktura"></a>

Вы кладёте удочки в одинарные сундуки, а улов забираете из двойного сундука. Улов из Рыбной фермы используется для работы [Торгового корабля](torgovyi-korabl.md).
