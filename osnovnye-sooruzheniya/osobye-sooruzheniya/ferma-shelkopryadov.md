# Ферма шелкопрядов

| Требуемая технология | Загонная охота |
| -------------------- | -------------- |
| Стоимость            | 25,000 монет   |
| Молоточки            | 1,600          |
| Прочность (HP)       | 1,000          |
| Очки города          | 900            |

Ферма шелкопрядов - [новое](../../obshaya-informaciya/novosti-obnovlenii/obnovlenie-1.2-dobro-pozhalovat-v-danzh.md) специализированное здание, позволяющее жителям города покупать нитки по фиксированной цене.&#x20;

{% hint style="warning" %}
64 нитки = 4,096 монет.
{% endhint %}
