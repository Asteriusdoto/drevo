# Верфь

| Размер               | **2х2**                |
| -------------------- | ---------------------- |
| Необходимое здание   | **-**                  |
| Требуемая технология | **Мореплавание**       |
| Стоимость            | **35,000 монет**       |
| Налог                | **1,500 монет /сутки** |
| Молоточки            | **1500**               |
| Прочность            | **300**                |
| Очки                 | **1300**               |
| Ограничение (кол-во) | **1**                  |

### Описание <a href="#opisanie" id="opisanie"></a>

Верфь является необходимым промежуточным зданием для постройки [Торгового Корабля](../osobye-sooruzheniya/torgovyi-korabl.md).

***

### Особенности <a href="#osobennosti" id="osobennosti"></a>

* Может быть построено только в чанках [океана, реки или пляжа!](../../osnovnye-kharakteristiki/kultura.md#bonusy-ot-biomov)

***
