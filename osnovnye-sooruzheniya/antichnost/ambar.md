# Амбар

| Размер               | 1x1                    |
| -------------------- | ---------------------- |
| Размер               | **1х1**                |
| Необходимое здание   | **-**                  |
| Требуемая технология | **Керамика**           |
| Стоимость            | **2,500 монет**        |

| Молоточки            | **140**                |
| Прочность            | **50**                 |
| Очки                 | **250**                |
| Ограничение (кол-во) | **-**                  |

### Описание <a href="#opisanie" id="opisanie"></a>

Амбар - это структура, созданная специально для [Коттеджей](../osobye-sooruzheniya/kottedzh.md).

{% hint style="info" %}
Коттедж каждый час забирает еду из Амбара, повышает свой уровень, и платит монеты в казну города.
{% endhint %}

В Амбаре расположена табличка, в которую нужно ложить <mark style="color:$success;">еду</mark>, иначе коттеджи не будут вырабатывать монеты.

***

#### Дополнительная информация <a href="#dopolnitelnaya_informaciya" id="dopolnitelnaya_informaciya"></a>

В Амбар нужно класть следующие предметы:

* Хлеб
* Морковь
* Картофель
* Сырая свинина (На 5 и 6 уровнях [Коттеджа](../osobye-sooruzheniya/kottedzh.md))
