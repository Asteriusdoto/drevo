| -------------------- | -------------------- |

### Описание <a href="#opisanie" id="opisanie"></a>

Бакалея - это структура, которая предоставляет возможность <mark style="color:$success;">купить еду</mark>. Всего предлагается 5 видов пищи. Чтобы разблокировать все виды потребуется улучшить Бакалею. Сделать это можно командой <mark style="color:$warning;">/town upgrade buy</mark> Бакалея.

***

#### Покупка еды <a href="#pokupka_edy" id="pokupka_edy"></a>

Для покупки еды нажмите на нужную табличку с едой которую хотите купить, но убедитесь, что улучшили бакалею до нужного уровня!

***

### Команды <a href="#komandy" id="komandy"></a>

{% hint style="info" %}
<mark style="color:$warning;">/town upgarade buy Бакалея</mark> \[I-V] - улучшает Вашу бакалею

<mark style="color:$warning;">/town set grocerfee</mark> \[5-15] - устанавливает пошлину для НЕ жителей Вашей цивилизации
{% endhint %}
