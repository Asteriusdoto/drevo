### Описание <a href="#opisanie" id="opisanie"></a>

Библиотека - это место, где можно наложить чары на инструменты, оружие или броню. Библиотека ограничена 4 чарами, поэтому мэр должен выбирать наиболее подходящие для своего города. Стоимость зачарования указана на табличке.

***

### Наложение чар <a href="#nalozhenie_char" id="nalozhenie_char"></a>

Для того, чтобы наложить чары на инструмент, нужно нажать с инструментом в руке по табличке с нужным зачарованием, но убедитесь, что библиотека улучшена до нужного уровня и улучшено само зачарование!

Доступные чары: Добыча 1-2, Эффективность 1-5, Удача 1, Приманка 1 и Шелковое касание. В Библиотеке можно купить SoulBound в виде зачарования, которое сохраняет эту вещь у Вас в инвентаре после смерти. Также доступна Невесомость 1, которая уменьшает урон от падения.

***

### Функции <a href="#funkcii" id="funkcii"></a>

**Выставление чар**

Только мэр может выбрать нужные чары, используя команду: <mark style="color:$warning;">/t upgrade buy</mark>

### Команды <a href="#komandy" id="komandy"></a>

{% hint style="warning" %}
<mark style="color:$warning;">**/t upgrade list library**</mark> - Показывает список доступных улучшений\
<mark style="color:$warning;">**/t upgrade buy**</mark> \[Зачарование]\[Уровень зачарования (Римскими цифрами)] - Купить зачарование для библиотеки\
<mark style="color:$warning;">**/t upgrade buy Библиотека**</mark> \[II - IV] - Повысить уровень Библиотеки\
<mark style="color:$warning;">**/t set libraryfee**</mark> \[5-15] - Изменить размер пошлины для Библиотеки\
<mark style="color:$warning;">**/t reset library**</mark> - Удаляет все зачарования из Библиотеки
{% endhint %}
