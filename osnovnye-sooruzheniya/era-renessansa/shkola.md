# Школа

| Размер               | **1х1**                |
| -------------------- | ---------------------- |
| Необходимое здание   | **Университет**        |
| Требуемая технология | **Научный метод**      |
| Стоимость            | **75,000 монет**       |
| Налог                | **1,500 монет /сутки** |

### Описание <a href="#opisanie" id="opisanie"></a>

Школа - это научная структура, которая генерирует [науку](../../osnovnye-kharakteristiki/nauka/) для города. Школу можно построить, только если в городе уже построен [Университет](../srednie-veka/universitet.md).
