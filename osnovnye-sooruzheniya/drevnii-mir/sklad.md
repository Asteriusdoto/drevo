|                      |                    |
| -------------------- | ------------------ |

### Описание <a href="#opisanie" id="opisanie"></a>

Склад - это важное строение, которое обеспечивает работу шахт, которые потребляют редстоун. На складе есть 8 двойных сундуков, в которые Вы должны класть <mark style="color:red;">редстоун</mark>. Поверьте, этого хватит надолго.

Подробнее про потребление редстоуна можно почитать в разделе [Шахта](../osobye-sooruzheniya/shakhta.md).
