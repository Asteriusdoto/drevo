# Монумент

| Размер               | **1х1**                |
| -------------------- | ---------------------- |
| Необходимое здание   | **-**                  |
| Требуемая технология | Не требует технологии  |
| Стоимость            | **10,000 монет**       |
| Налог                | **1,000 монет /сутки** |

| Прочность            | **250**                |
| Очки                 | **700**                |

### Описание <a href="#opisanie" id="opisanie"></a>

Монумент - это культурная структура, которая генерирует [<mark style="color:purple;">Культуру</mark> ](../../osnovnye-kharakteristiki/kultura.md)и [<mark style="color:yellow;">Счастье</mark>](../../osnovnye-kharakteristiki/schaste.md) для Вашего города. Город будет быстрее развиваться благодаря монументу, поэтому строить его рекомендуется в первую очередь.
