# Радиовышка

| Размер               | **1х1**                |
| -------------------- | ---------------------- |
| Необходимое здание   | **-**                  |
| Требуемая технология | **Радио**              |
| Стоимость            | **275,000 монет**      |
| Налог                | **2,000 монет /сутки** |
| Молоточки            | **7500**               |
| Прочность            | **1,500**              |
| Очки                 | **6000**               |
| Ограничение (кол-во) | **1**                  |

### Описание <a href="#opisanie" id="opisanie"></a>

Радиовышка - это структура поздних этапов игры, которая генерирует [<mark style="color:purple;">культуру</mark>](../../osnovnye-kharakteristiki/kultura.md) для города, а также позволяет быстрее переходить из одного государственного строя в другой.

***

### Модификации и Эффекты <a href="#modifikacii_i_effekty" id="modifikacii_i_effekty"></a>

Культура: <mark style="color:purple;">+375</mark>

Время анархии при переходе цивилизации из одного государственного строя в другой <mark style="color:$success;">-50%</mark>

{% hint style="warning" %}
Внимание! Работает только в случае если достроенные радиовышки стоят в каждом родном/купленном/присоединенном городе цивилизации.
{% endhint %}
