# Деньги

Деньги - топливо для развития вашей Цивилизации. Вы можете потратить их на исследование технологий, постройку зданий, покупку ресурсов, зачарования предметов, налоги и т.д.

Увидеть количество доступных монет можно прописав команду <mark style="color:$warning;">**/money**</mark>.

Передать деньги другому игроку можно командой <mark style="color:$warning;">**/pay**</mark> <Ник> <Сумма>.

***

### Заработок монет <a href="#zarabotok_monet" id="zarabotok_monet"></a>

#### 1. Поход в шахту <a href="#id-1._pokhod_v_shakhtu" id="id-1._pokhod_v_shakhtu"></a>

Кроме незначительных денег за выкапывание угля, красной пыли, алмазов и пр., есть более хороший способ подзаработать денег.

Вы можете продать ваше: железо, золото, алмазы и изумруды в банке на спавне, в своём банке или в банке других цивилизациях. Чем выше уровень банка, тем больше денег вы получите при продаже. Если вы продаёте не в своём банке, то с полученных денег взымается налог, уходящих в казну хозяина банка.

| Плюсы                         | Минусы                     |
| ----------------------------- | -------------------------- |
| Простое для фанатов майнкрафт | Небольшой доход            |
| Булыжник как побочный продукт | Опасно потерять всё в лаве |

#### 2. Фарм в данже <a href="#id-2._farm" id="id-2._farm"></a>

Мобы в игре делятся на 4 уровня: <mark style="color:$success;">**т1**</mark>, <mark style="color:$warning;">**т2**</mark>, <mark style="color:purple;">**т3**</mark>, <mark style="color:red;">**т4**</mark>. В соответствии с ними существует 4 типа оружия и брони. Монстров более высокого уровня сложнее убить, но с них падает больше денег и больше ценных ресурсов. Оружие может наносить урон монстрам, уровень которых НЕ больше, чем у оружия +1. То есть, что бы убить Т3 Зомби вам понадобится меч т2 или выше.

Больше всего монет выпадает с зомби, но они и самые опасные.

Подробнее про данж [тут](../obshaya-informaciya/danzh.md).

| Плюсы                            | Минусы                               |
| -------------------------------- | ------------------------------------ |
| Много денег                      | Опасность от других игроков в данже. |
| Есть чем заняться жителям города | Нужны ресурсы на мечи и броню        |
| Ресурсы как побочный продукт     | Нужны деньги на починку оружия       |

#### 3. Дробилка <a href="#id-3._drobilka" id="id-3._drobilka"></a>

После изучения начальной технологии "Шахты" появляется возможность построить здание [Дробилка](../osnovnye-sooruzheniya/osobye-sooruzheniya/drobilka.md). В нём вы можете переработать камень в гравий с неплохим шансом выпадения ценных ресурсов (золото, алмазы и изумруды). Эти ресурсы в свою очередь можно продавать в Банке.

Для повышения эффективности необходимо построить структуру [Карьер](../osnovnye-sooruzheniya/osobye-sooruzheniya/karer.md). Если положить в определённый сундук карьера кирки, то он будет выкапывать булыжник и другие ресурсы, которые можно добыть в шахте. Булыжник складываете в дробилку, ресурсы продаёте в банке.

Связку Карьер-Дробилка можно сделать эффективнее с помощью государственного строя [Деспотизм](../upravlenie-gorodom-i-civilizaciei/gosudarstvennyi-stroi.md#svodnaya_tablica_vsekh_politicheskikh_stroev).

| Плюсы                                                  | Минусы                                                           |
| ------------------------------------------------------ | ---------------------------------------------------------------- |
| Дешёвые технологии                                     | Количество карьеров ограничено количеством городов в цивилизации |
| Деспотизм имеет небольшой налог на содержание построек | Нужен регулярный онлайн (карьеры наполняются за 40-50 минут)     |
| Счастье никак не влияет на доход                       | Для повышения дохода нужно постоянно улучшать банк               |
| Побочный продукт железо, уголь, красная пыль           |                                                                  |
| Требует очень мало слотов улучшения города             |                                                                  |
| Средний уровень дохода                                 |                                                                  |

#### 4. Коттеджи <a href="#id-4._kottedzhi" id="id-4._kottedzhi"></a>

Самый распространённый способов заработка основан на связке сооружений [Ферма](../osnovnye-sooruzheniya/osobye-sooruzheniya/ferma.md) - [Амбар](../osnovnye-sooruzheniya/antichnost/ambar.md) - [Коттедж](../osnovnye-sooruzheniya/osobye-sooruzheniya/kottedzh.md).

После изучения всех требуемых технологий, начинаем постройку Фермы. Далее строим Амбар. Недалеко от фермы, чтобы носить еду было быстрее. Затем строим Коттеджи. Чем больше коттеджей, тем больше денег с них вы получите.&#x20;

В Амбар нужно класть: Сырой картофель, морковь, хлеб. Коттеджи раз в час берут еду из амбара, взамен генерируют монеты в казну города и повышают свой уровень. Чем больше уровень коттеджа, тем больше он потребляет еды и больше приносит монет в час.

Коттеджи и фермы являются особыми постройками и <mark style="color:$warning;">**требуют слот**</mark> для постройки.

[Государственный строй](../upravlenie-gorodom-i-civilizaciei/gosudarstvennyi-stroi.md) "Монархия" и технология "Налоги" увеличивают доход с коттеджей почти в 4 раза. На доход очень сильно влияет уровень счастья в городе.

| Плюсы                                           | Минусы                                           |
| ----------------------------------------------- | ------------------------------------------------ |
| Небольшая стоимость технологий                  | Нужно строить автофермы                          |
| Доход в каждом городе с коттеджами              | Использует все слоты улучшения города            |
| Значительный доход на больших уровнях коттеджей | Счастье сильно влияет на доход                   |
|                                                 | Нужно следить за фермой и наполняемостью амбаров |
|                                                 | Монархия имеет большой налог на постройки        |

#### 5. Торговые судна <a href="#id-5._torgovye_sudna" id="id-5._torgovye_sudna"></a>

Доход основан на связке построек [Рыбная ферма](../osnovnye-sooruzheniya/osobye-sooruzheniya/rybnaya-ferma.md) - [Торговое судно](../osnovnye-sooruzheniya/osobye-sooruzheniya/torgovyi-korabl.md). После изучения технологии "Навигация" будет доступно строение Торговое судно, которое, кроме дохода, будет генерировать дополнительную культуру, молоточки и растительность. После изучения технологии "Механизмы" нужно построить Рыбную ферму, которая потребляет удочки и взамен дает рыбу. Торговое судно каждый час забирает рыбу и дает минералы, порох и ресурсы, требуемые для создания катализаторов.

| Плюсы                          | Минусы                                             |
| ------------------------------ | -------------------------------------------------- |
| Генерирует культуру            | Нужно пополнять Рыбные фермы удочками              |
| Средний доход                  | Нужно пополнять Торговое судно рыбой               |
| Вместе с деньгами дает ресурсы | Рыбные фермы занимают большое количество Улучшений |

Увеличить доход от торговых кораблей могут: постройка [маяк](../osnovnye-sooruzheniya/antichnost/mayak.md), чудеса света [Александрийский маяк](../chudesa-sveta/aleksandriiskii-mayak.md) и [Флагман Ингерманланд](../chudesa-sveta/flagman-ingermanland.md).
